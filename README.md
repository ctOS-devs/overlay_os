## Основная идея
Идея в том, чтобы запускать ЛЮБОЕ ПО не используя root, не модифицируя установленный Linux.
На тех же роутерах, ТВ-приставках, ... Где можно получить ssh доступ.

Требования:
1. Установленный Linux x86/aarch64.
2. Доступ по ssh.
3. Консоль sh/bash/...
4. Место на диске/флешке/...

Ключевые принципы:
1. Никак не модифицировать оригинальную ОС.
2. Работать без рута (как минимум 80% вещей)
3. Работать где угодно: /home/user, USB, HD, RAM, ..., ?загрузка по сети?
